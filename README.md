# Donation Calculator

An interactive donation calculator designed to help nonprofit organizations clearly communicate the real-world impact of donations. This project focuses on user experience, transparency, and engagement to encourage meaningful contributions.

## Features
- Real-time donation impact calculations
- Clean, mobile-friendly UI
- Simple and intuitive user interaction
- Easy to embed on a website (static front-end)
- Optional payment button support (keys not included)

## Tech Stack
- HTML
- CSS
- JavaScript

## Live Demo
Live version (GitHub Pages):  
https://tygreen1.github.io/Donation_Calculator/

## Security & Privacy
This repository is intended for portfolio/demo use.
- No server-side secrets should be stored in this repo.
- Any API keys, payment credentials, or private endpoints should be removed or replaced with placeholders.
- If a payment SDK is used, only publishable/client-side configuration should appear here.

## Project Goal
Help organizations increase donor confidence and conversion by showing how donation amounts translate into tangible impact.
